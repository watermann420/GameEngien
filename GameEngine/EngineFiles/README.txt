Place project assets here (Blender-ready). Export FBX from Blender with:
- Apply transforms (Ctrl+A), set Unit Scale 1.0, Forward -Z, Up Y, Triangulate.
- Embed or copy textures into `EngineFiles/textures/` and keep relative paths.

Structure:
- meshes/    -> FBX/OBJ/GLTF exports from Blender (1:1 scale)
- audio/     -> WAV/MP3/etc
- video/     -> MP4/etc
- textures/  -> PNG/JPG/etc (albedo)
